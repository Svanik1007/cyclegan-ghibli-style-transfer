# CycleGAN Ghibli Style Transfer

A CycleGAN implementation in PyTorch that transforms real-world photos into Ghibli-style artwork.

## Overview

This project uses CycleGAN to learn image-to-image translation between:

- Real photographs
- Ghibli-style images

without requiring paired training examples.

## Technologies

- Python
- PyTorch
- Google Colab
- CycleGAN

## Training Details

- Dataset: ~2500 photos + ~2500 Ghibli images
- Resolution: 128×128
- Epochs: 25
- GPU: Tesla T4
